### 响应式编程：(Reactive programing)

1. 关心数据流和流的变化。
2. 和传统方式的区别：有数据即刻响应
3. 设计模式：观察者模式
4. 关键机制：背压。数据流的生产端能够知道消费端的处理能力，并以此调整生产量。
5. 核心项目：Reactor。flux(publisher)和 mono(publisher)
6. 一些实现框架：Spring-webflux.
